# Accounting Practice Trainer

Standalone Next.js app for the accounting Midterm 2 trainer.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` or `http://localhost:3000/accounting`.

## Deploy to Vercel

```bash
npx vercel login
npx vercel --prod
```

No extra environment variables are required for the accounting trainer.
